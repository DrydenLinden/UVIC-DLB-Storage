Faculty and Department Preferences

For the following faculties, please select all that interest you. Click on the expand button to select specific departments in the faculty. If no specific departments are selected within a chosen faculty, all departments will be considered. If left blank, other criteria on this form will be used to generate suggested courses, and all faculties will be given equal priority. Please note that despite having preferences in departments of your own, if any courses of those courses are restricted to students registered in that department, those courses will not be suggested. Only courses which you can register in for this current semester will be suggested.
